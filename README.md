# hwaya_bot
## Features
1. 新增/刪除食物
2. 每日定時提醒使用者是否有
  a. 當天過期之食物 (提示完將會自動刪除)
  b. 三天內要過期之食物
## Command
1. /註冊 : 使用者必須先註冊才能使用新增/刪除等功能
2. /新增 : 新增食物
  a. /新增：[物品]～[月日]@[地點]
3. /刪除
  a. /刪除：[物品]～[月日]@[地點]
4. /help : 顯示目前支援的指令
5. /howto : 顯示指令規則
6. /id : 顯示使用者自己的ID
7. /list : 列出已儲存之食物
8. /location : 列出已登錄的地點
9. /code : 程式碼相關
